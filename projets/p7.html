Exploitation d'une base de données

Contexte

Le but de ce projet est de comparer les performances de différents algorithmes de tri. Pour cela, nous avons implémenté les algorithmes suivants :

  • Tri à bulles
  • Tri par sélection
  • Tri par insertion
  • Tri rapide

Objectifs

Les objectifs de ce projet sont les suivants :

  • Implémenter les algorithmes de tri
  • Comparer les performances des algorithmes
  • Écrire un rapport sur les résultats obtenus

Résultats

Les résultats de notre étude montrent que le tri rapide est le plus performant des algorithmes testés. En effet, il est capable de trier un tableau de 1 million d'entiers en moins de 1 seconde, alors que les autres algorithmes prennent plusieurs minutes pour effectuer la même tâche.
